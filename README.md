#### Example Function 2: Find the maximum value in a list

```python
def find_max(numbers):
  max_value = numbers[0]
  for num in numbers:
    if num > max_value:
      max_value = num
  return max_value
```

#### Example Function 3: Check if a string is a palindrome

```python
def is_palindrome(string):
  reversed_string = string[::-1]
  if string == reversed_string:
    return True
  else:
    return False
```
